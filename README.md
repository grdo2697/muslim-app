# 🕌 تطبيق مسلم - Muslim App

تطبيق وموقع إسلامي شامل متكامل للعالم - **موقع + تطبيق جوال**

## ✨ الميزات الرئيسية

### 📱 القسم الأساسي
- ⏰ **مواقيت الصلاة** - في كل دول العالم مع التحديث التلقائي
- 🔔 **الأذان** - ينبه حتى مع إطفاء الشاشة مع خيار الإيقاف
- 📖 **القرآن الكريم** - قراءة + استماع + تفسير
- 🎤 **القصائد الحسينية** - مع التلاوات الصوتية
- 🖼️ **صور أهل البيت** - معرض متقدم
- 🤲 **الأذكار والادعية** - منظمة حسب الأوقات
- 📅 **المناسبات الإسلامية** - مع تنبيهات وبرامج يومية

### 🎓 قسم التعليم
- 📚 **شرح طريقة الصلاة** - خطوة بخطوة مع فيديوهات
- 💧 **شرح الوضوء** - خطوات مفصلة مع صور
- 👶 **تعليم الأطفال** - نسخة مبسطة من التطبيق
- 🎓 **دروس دينية** - محاضرات من العلماء

### 🌙 قسم العبادات المتقدم
- 🤲 **أحكام الصلاة والوضوء**
- 🕌 **صلوات مستحبة** - الليل، الاستسقاء، الكسوف
- 🕯️ **ليالي القدر** - عدادات تنازلية
- 🛡️ **أدعية الحماية والسلام**

### 👥 قسم المجتمع
- 💰 **نظام التبرعات** - مساعدة الأيتام والفقراء والطلاب
- 📊 **متابعة التبرعات** - رؤية تأثير مساهمتك
- 🏆 **برنامج المكافآت** - نقاط مقابل الأنشطة
- 💬 **المنتدى** - التواصل مع المجتمع

### 🔧 الإعدادات والحسابات
- 👤 **نظام المستخدمين** - تسجيل دخول آمن
- 🔐 **الحساب الشخصي** - حفظ التفضيلات
- 📱 **مزامنة سحابية** - بين الجهاز والموقع
- 🌙 **وضع ليلي**
- 🌍 **لغات متعددة** - العربية، الإنجليزية، الفارسية، الأردية

## 🛠️ التقنيات المستخدمة

### **Frontend (الموقع)**
- React.js + TypeScript
- Tailwind CSS (تصميم)
- Vite (بناء سريع)

### **Mobile (التطبيق)**
- React Native
- Expo (توزيع سهل)

### **Backend (الخادم)**
- Node.js + Express
- Firebase (قاعدة البيانات والمصادقة)
- Cloud Functions

### **الخدمات الخارجية**
- Aladhan API (مواقيت الصلاة)
- Firebase Notifications (الإشعارات)
- Google Cloud Storage (الملفات)

## 🚀 البدء السريع

### **متطلبات التثبيت**
```bash
- Node.js v18+
- npm أو yarn
- Firebase CLI
- Expo CLI (للتطبيق الجوال)
```

### **التثبيت**
```bash
# استنساخ المستودع
git clone https://github.com/grdo2697/muslim-app.git
cd muslim-app

# تثبيت المكتبات
npm install
```

### **تشغيل المشروع**
```bash
# الموقع
cd frontend && npm start

# الخادم
cd backend && npm start

# التطبيق الجوال
cd mobile && npm start
```
